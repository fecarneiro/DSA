#

## Passagem por referência:

- Objetos e Arrays
- Recebe uma cópia referência idêntica ao valor de memória original
- Afeta original

## Passagem por valor:

- Número, Valor, String
- Cópia independente
- Não afeta o original (endereço de memória diferente)

## Arrays

arr[1, 2, 3, 4, 5, 6]

- splice(index1, index2): para remover elementos do indice 1 AO 2
- pop()
- push()
- shift() : deleta o primeiro num
- unshift() : add número no início
